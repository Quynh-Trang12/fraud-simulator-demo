# Dataset Setup Instructions

To maximize points for "Data Collection", please download the following datasets and place them in this `data/` directory.

## 1. Primary Dataset: Online Payments Fraud Detection

- **Source**: [Kaggle Link](https://www.kaggle.com/datasets/rupakroy/online-payments-fraud-detection-dataset/data)
- **File Name**: `onlinefraud.csv` (Renamed from whatever the download gives, usually `PS_2017...csv` or similar. Please verify the content matches).
- **Description**: Mobile money transactions. Features: `step`, `type`, `amount`, `oldbalanceOrg`, `newbalanceOrig`...

## Final Directory Structure

```text
/data
```
